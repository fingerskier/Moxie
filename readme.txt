Moxie is an issue tracker for ColdFusion applications

Allows devs and client users to define issues, attach supporting documents, and discuss progress.  Devs can send announcements per issue.


Features:
	Manage dev/client users.
	Devs & clients can raise issues.
	Users can upload attachments (documents, images, etc.)
	Devs can send announcements regarding issues (email, SMS)
